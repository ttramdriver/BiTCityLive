# BiTCityLive
Aplikacja mobilna / [internetowa](https://bitcitylive.jablecznik.xyz), która wyświetla dane (czas odjazdu, czas opóźnienia, itp. (planowane też jest położenie na mapie)) o zbiorkomie, tj. komunikacji miejskiej (tramwaje, autobusy) w Bydgoszczy, Toruniu, oraz tam gdzie dojeżdżają linie międzygminne obu miast oraz pociągów w całym Kujawsko-Pomorskim.
Aplikacja internetowa znajduje się pod adresem https://bitcitylive.jablecznik.xyz.
Wszystkie dane zbierane są z http://odjazdy.zdmikp.bydgoszcz.pl oraz http://sip.um.torun.pl:8080. 
Podziękowania dla [Yaoofi](https://github.com/Yaoofi) za pomoc w odjazdach na żywo pociągów.
Work in Progress.
