# BiTCityLive
Aplikacja / strona, która wyświetla dane (czas odjazdu, czas opóźnienia, położenie na mapie itp.) o zbiorkomie w Bydgoszczy, Toruniu i pomiędzy nimi. Wlicza to: komunikację miejską w obu miastach (tramwaje, autobusy) oraz pociągi na trasach pomiędzy nimi. Wszystkie dane zbierane są z http://odjazdy.zdmikp.bydgoszcz.pl oraz http://sip.um.torun.pl:8080
Work in Progress.
